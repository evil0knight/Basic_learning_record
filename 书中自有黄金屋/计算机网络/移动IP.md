# 移动IP

[← 返回 MOC](MOC.md) | [← 主页](../../index.md)

---

移动站有一个永久地址（归属地址），归属代理负责管理。

当移动站漫游到外地网络时，向外地代理登记，获得一个**转交地址**（外地地址）。

## 数据转发流程

**发送给移动站：**

1. 数据报先发到移动站的永久地址
2. 归属代理截获，用**IP隧道**（IP套娃）将数据报封装后转发到转交地址
3. 外地代理解封装，通过 MAC 地址直接转发给移动站

**移动站发送数据：**

- 移动站直接发送，不需要经过归属代理转发

## 关键点

- 一个外地代理的转交地址可被多个移动站共用
- 外地代理用 MAC 地址在本地网络中转发数据帧
