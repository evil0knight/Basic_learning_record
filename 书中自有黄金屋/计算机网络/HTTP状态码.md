---

| 类别 | 含义 | 常见状态码 | 速记 |
| --- | --- | --- | --- |
| [1xx](#1xx-信息) | 请求已收到 | `100` | 继续发 |
| [2xx](#2xx-成功) | 请求成功 | `200` `201` `204` | 成了 |
| [3xx](#3xx-重定向) | 需要额外操作 | `301` `302` `304` | 去别处/用缓存 |
| [4xx](#4xx-客户端错误) | 请求有问题 | `400` `403` `404` | 你错了 |
| [5xx](#5xx-服务器错误) | 服务器出错 | `500` `502` `503` `504` | 服务器错了 |

## 1xx 信息

| 状态码 | 含义 |
| --- | --- |
| `100 Continue` | 继续发送请求体 |

## 2xx 成功

| 状态码 | 含义 |
| --- | --- |
| `200 OK` | 请求成功 |
| `201 Created` | 资源已创建 |
| `204 No Content` | 成功,但无返回体 |

## 3xx 重定向

| 状态码 | 含义 |
| --- | --- |
| `301 Moved Permanently` | 永久重定向 |
| `302 Found` | 临时重定向 |
| `304 Not Modified` | 资源未改,用缓存 |

## 4xx 客户端错误

| 状态码 | 含义 |
| --- | --- |
| `400 Bad Request` | 请求报文有误 |
| `403 Forbidden` | 服务器拒绝访问 |
| `404 Not Found` | 资源不存在 |

## 5xx 服务器错误

| 状态码 | 含义 |
| --- | --- |
| `500 Internal Server Error` | 服务器内部错误 |
| `502 Bad Gateway` | 网关或代理错误 |
| `503 Service Unavailable` | 服务暂不可用 |
| `504 Gateway Timeout` | 网关超时 |
