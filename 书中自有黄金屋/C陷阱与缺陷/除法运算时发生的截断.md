
# 除法运算时发生的截断

#### 向零取整 (Truncate toward zero)

先不管正负号，直接算绝对值 **$3 / 2 = 1.5$**，然后把小数位砍掉，得到 **$1$**。最后把负号补回来，商就是  **$-1$** 。

* **商 (**$q$**)** ：**$-1$**
* **推导余数 (**$r$**)** ：根据公式 **$-3 = (-1) \times 2 + r$**，得出 **$r = -1$**。
* **特点** ：余数的符号总是和被除数相同。

整数除法必须采用  **向零取整 (Truncation toward zero)** 。

**结果** ：现在无论你在什么机器上跑，`-3 / 2`  **必须等于 **$-1$**** ，`-3 % 2` **必须等于 **$-1$****
