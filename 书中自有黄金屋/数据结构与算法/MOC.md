## 高频考点 / 复杂度速查

| 结构 / 算法 | 查找 | 插入 | 删除 | 空间 |
|-------------|------|------|------|------|
| 数组（有序） | O(log n) | O(n) | O(n) | O(n) |
| 链表 | O(n) | O(1) | O(1) | O(n) |
| 哈希表 | O(1) avg | O(1) avg | O(1) avg | O(n) |
| 二叉搜索树 | O(log n) avg | O(log n) avg | O(log n) avg | O(n) |
