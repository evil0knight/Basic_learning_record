## 换算规则说明

- **官方原价**：Anthropic 官网公布的美元定价（$/M token）
- **1元1刀无倍率**：中转站充值比例为 1元人民币 = 1美元，无加价倍率
- **换算成实际美元**：按当前汇率 1元 ≈ $0.1457（1美元 = 6.8649元，2026年4月9日），将中转站¥价格换算为真实美元成本
