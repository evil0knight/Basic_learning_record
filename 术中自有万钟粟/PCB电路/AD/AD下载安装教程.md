## 一、软件安装

1. 右键下载好的**Altium Designer 17**压缩包，选择 **解压到当前文件夹** 。
2. 打开解压文件夹，双击运行**AltiumDesignerSetup_17_0_6**安装程序。
3. 点击 **Next** ，语言选择 **Chinese** ，勾选 **I accept the agreement** ，再点 **Next** 。
4. 持续点击 **Next** ，点击**Default**修改安装路径（ **路径不可含中文** ），继续点 **Next** 。
5. 等待软件自动安装，安装完成后 **取消勾选 Run Altium Designer** ，点击 **Finish** 。
6. 打开开始菜单，找到**Altium Designer**图标，拖动到桌面创建快捷方式。

## 二、许可激活

1. 双击桌面图标启动软件，点击 **OK** ，关闭右上角**Messages**界面。
2. 点击 **Add standalone license file** ，选择解压文件夹中**licenses**目录下任意文件，点击**打开**完成激活。

## 三、汉化与收尾

1. 点击 **DXP** → **Preference** ，进入**General**选项，勾选 **Use localized resources** ，连续点击 **OK** 。
2. 重启软件，**Altium Designer 17**安装、激活、汉化全部完成。

注意激活文件选择完成之后要重启才能生效
