## 基本理解

电容最直观的作用是存电场能量。它不擅长长期供电，但很擅长在很短时间里补电流、压纹波、旁路高频噪声。

几个最常用的关系：

- $Q = C \cdot U$
- $i = C \frac{du}{dt}$
- $X_C = \frac{1}{2 \pi f C}$
- $E = \frac{1}{2} C U^2$

可以先记两句：

- 电容两端电压不能突变。
- 频率越高，电容的阻抗通常越小，但真实电容到了更高频还会受 ESL 限制。

## 常看参数

| 参数     | 作用                             |
| -------- | -------------------------------- |
| 容值     | 决定能存多少电、滤波有多强       |
| 耐压     | 至少要高于实际工作电压，别贴边用 |
| ESR      | 影响发热、纹波和瞬态表现         |
| ESL      | 影响高频效果，去耦时很关键       |
| 容差     | 实际值和标称值不完全一样         |
| 漏电流   | 保持电压能力差时会更明显         |
| 纹波电流 | 电源输入输出处必须看             |

## 板子里常见用法

- 去耦：芯片电源脚旁边放小电容，处理高频电流脉冲。
- 滤波：输入输出放较大电容，压低纹波，稳住母线。
- 储能：负载瞬时电流大时，先从电容拿一口电。
- RC：和电阻一起做延时、滤波、吸收尖峰。

板上最常见的误区不是“容值不够”，而是“电容离得远、回路太长、类型用错”。
