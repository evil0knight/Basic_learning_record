# 外设代码库文件模板

每个外设对应一个 `.md` 文件，结构如下：

---

# \<外设名\>程序

[← 返回 MOC](MOC.md) | [← 主页](../index.md)

---

## \<写法名称\>（\<一句话适用场景\>）

\<2-3句说明：这种写法的核心机制是什么，适合什么场景，有什么限制\>

```c
// 完整可复制的代码片段
// 包含：必要的全局变量声明、初始化函数、核心函数、main示例
```

---

## \<写法名称2\>（\<一句话适用场景\>）

...

---

## 规则

1. 每种写法必须有 `## 标题`，标题即选型菜单里显示的选项名
2. 代码块必须用 ` ```c ` 标注语言
3. 代码块内容要能直接复制使用（含必要的变量声明）
4. 平台相关的函数名（如 `HAL_UART_*`、`huart1`）保留原样，由用户适配
